Java Basics

Introduction to Java

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle). It's designed to be platform-independent and is widely used for building web applications, mobile apps (Android), and enterprise software.

Variables and Data Types

Java has several primitive data types:

  • int - Integer values
  • double - Floating-point numbers
  • boolean - True or false values
  • char - Single characters
  • String - Text sequences

Example: Hello World

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Control Structures

Java supports common control structures:

// If-else statement
if (x > 0) {
    System.out.println("Positive");
} else if (x < 0) {
    System.out.println("Negative");
} else {
    System.out.println("Zero");
}

// For loop
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// While loop
int count = 0;
while (count < 5) {
    System.out.println(count);
    count++;
}

Arrays

Arrays in Java are fixed-size collections:

// Declare and initialize
int[] numbers = {1, 2, 3, 4, 5};

// Access elements
int first = numbers[0];

// Array length
int size = numbers.length;
